### Suggested `setup_packages.R` Content

For maximum reproducibility, you can use a script like this to ensure all packages are installed and loaded:

``` r
# List of required packages
required_packages <- c(
    "dplyr",      # Data manipulation
    "tidyr",      # Data tidying
    "ggplot2",    # Visualization
    "igraph",     # Network analysis
    "readr",      # Fast CSV reading
    "lmtest"      # Statistical tests (if used for regression diagnostics)
)

# Function to check, install, and load packages
install_and_load <- function(packages) {
    for (pkg in packages) {
        if (!require(pkg, character.only = TRUE)) {
            install.packages(pkg, dependencies = TRUE)
            library(pkg, character.only = TRUE)
        }
    }
}

# Execute the function
install_and_load(required_packages)

# Ensure R's core 'stats' package is available for Kruskal-Wallis/Wilcoxon tests
# (It's loaded by default, but good to be aware)

```
